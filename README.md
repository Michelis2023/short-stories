## 🎯 Περιγραφή Έργου

Η συλλογή αυτή περιλαμβάνει γοτθικά διηγήματα τρόμου που αντλούν έμπνευση από:

- 🦇 **Ελληνική Λαογραφία**: Ιστορίες γιαγιάδων, λαϊκοί μύθοι και παραδόσεις
- 🕯️ **Edgar Allan Poe**: Γοτθικό ύφος, ψυχολογική τάση και ατμόσφαιρα
- ⛪ **Βυζαντινή και Ιστορική Ελλάδα**: Θρησκευτικά μυστήρια και ιστορικά σκηνικά
- 🏥 **Συγχρονη ελληνική γλώσσα**: Παραδοσιακό ύφος με σύγχρονη αφήγηση

Ο ιστότοπος παρουσιάζεται με γοτθική αισθητική (σκοτεινά χρώματα, παλαιωμένο χαρτί, Cinzel και EB Garamond fonts) και είναι βελτιστοποιημένος για όλες τις συσκευές.

## 📚 Περιεχόμενα Συλλογής

Η συλλογή περιλαμβάνει **6 γοτθικά διηγήματα**:

### 1. 🦇 [Τα Σμυρδάκια](https://michelis2023.github.io/short-stories/ta-smyrdakia.html)
**Είδος:** Γοτθικό Τρόμου | **Ημ/νία:** Φεβρουάριος 2026  
Ένα σκοτεινό διήγημα για παιδιά που θάφτηκαν ζωντανά στα βάθη του βουνού και επιστρέφουν να διεκδικήσουν όσους τα ξέχασαν. Εμπνευσμένο από την ελληνική λαϊκή παράδοση.

### 2. 👻 [Η Σκιά Μέσα Στον Καθρέφτη](https://michelis2023.github.io/short-stories/i-skia-mesa-ston-kathrefthi.html)
**Είδος:** Υπερφυσικό Τρόμου | **Ημ/νία:** Φεβρουάριος 2026  
Στην Κωνσταντινούπολη του 1892, ένας έμπορος ανακαλύπτει ότι η αντανάκλασή του στον καθρέφτη έχει δική της βούληση. Μια ιστορία για την ταυτότητα, τον φόβο και την απώλεια του εαυτού.

### 3. ✋ [Το Χέρι Κάτω από το Χώμα](https://michelis2023.github.io/short-stories/to-xeri-kato-apo-to-xoma.html)
**Είδος:** Ψυχολογικό Τρόμου | **Ημ/νία:** Ιανουάριος 2026  
Ιστορία υπερφυσικού φόβου και παραμονής του παρελθόντος. Ένα χέρι που ξεπροβάλλει από το χώμα και δεν σε αφήνει να ξεφύγεις από τις πράξεις σου.

### 4. ⛪ [Το Ζοφερόν Μυστικόν του Αγίου Ταξιάρχη](https://michelis2023.github.io/short-stories/to-zopheron-mystikon-tou-agiou-taxiarchi.html)
**Είδος:** Θρησκευτικό Τρόμου | **Ημ/νία:** Ιανουάριος 2026  
Μια ιστορία τρόμου και υπερφυσικού που εξελίσσεται γύρω από έναν εγκαταλελειμμένο ναό και τα σκοτεινά μυστικά που κρύβει.

### 5. 🎭 [Η Λέσχη της Κυρίας Β.](https://michelis2023.github.io/short-stories/I-Leshi-tis-Kyrias-B..html)
**Είδος:** Γοτθικό Τρόμου | **Ημ/νία:** Δεκέμβριος 2025  
Ένα γοτθικό διήγημα τρόμου που εξιστορεί τα σκοτεινά μυστικά μιας αριστοκρατικής λέσχης και των μελών της.

### 6. 🏥 [Νυχτερινή Βάρδια στο Σανατόριο του Μαινάλου](https://michelis2023.github.io/short-stories/nukhterine-bardia-sto-sanatorio-tou-mainalou.html)
**Είδος:** Γοτθικό Τρόμου | **Ημ/νία:** Φεβρουάριος 2026  
Φθινόπωρο 1898, Σανατόριο Αγίας Ευφροσύνης στο Μαίναλο. Ένας νέος γιατρός σε νυχτερινή βάρδια ανακαλύπτει τα φρικτά μυστικά μιας κλειστής σοφίτας.

## 💻 Τεχνολογίες

**Απλό και διαφανές:**

- **HTML5** - Σημασιολογική δομή περιεχομένου
- **CSS3** - Γοτθική αισθητική και τυπογραφία (χωρίς frameworks)
- **Google Fonts** - Cinzel και EB Garamond για αυθεντική ατμόσφαιρα
- **GitHub Pages** - Δωρεάν φιλοξενία

**Δεν χρησιμοποιούνται:**

- ❌ JavaScript
- ❌ Βάσεις δεδομένων
- ❌ CSS frameworks (Bootstrap, Tailwind κ.λπ.)
- ❌ Build tools

Το project είναι εσκεμμένα **ελαφρύ**, **διαφανές** και **εύκολο στη συντήρηση**.

## 🚀 Τοπική Προβολή

Για να προβάλετε τη συλλογή τοπικά:

### 1. Κλωνοποίηση

```bash
git clone https://github.com/Michelis2023/short-stories.git
cd short-stories
```

### 2. Άνοιγμα στον Browser

**Επιλογή Α:** Απευθείας άνοιγμα
- Ανοίξτε το `index.html` στον browser σας

**Επιλογή Β:** Με local server (προτείνεται)

```bash
# Python 3
python -m http.server 8000

# ή Python 2
python -m SimpleHTTPServer 8000
```

Στη συνέχεια επισκεφθείτε: `http://localhost:8000/index.html`
