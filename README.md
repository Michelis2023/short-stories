## 📚 Διαθέσιμα Διηγήματα

**Περιγραφή**: Ένα σκοτεινό διήγημα για παιδιά που θάφτηκαν ζωντανά στα βάθη του βουνού και επιστρέφουν να διεκδικήσουν όσους τα ξέχασαν. Εμπνευσμένο από την ελληνική λαϊκή παράδοση και τις ιστορίες των γιαγιάδων γύρω από τη φωτιά.

### 👻 **Η Σκιά Μέσα Στον Καθρέφτη**
**Ημερομηνία**: Φεβρουάριος 2026 | **Είδος**: Υπερφυσικό Τρόμου  
**Περιγραφή**: Στην Κωνσταντινούπολη του 1892, ένας έμπορος ανακαλύπτει ότι η αντανάκλασή του στον καθρέφτη έχει δική της βούληση. Μια ιστορία για την ταυτότητα, τον φόβο και την απώλεια του εαυτού.
